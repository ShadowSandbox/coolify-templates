## Overview

Supabase is an open-source Firebase alternative built on Postgres. You get a full backend out of the box: database, authentication, auto-generated REST APIs, realtime subscriptions, and file storage — all self-hosted on your own server.

This template deploys the full Supabase stack: Studio, the Kong API gateway, GoTrue auth, PostgREST, Realtime, the Storage API, and Postgres itself.
