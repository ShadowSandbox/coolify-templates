## Overview

n8n is a fair-code workflow automation platform that combines a visual editor with the ability to write custom code. Connect 400+ apps, build AI agent workflows, and self-host everything so your data never leaves your server.

This template deploys n8n in queue mode with PostgreSQL for storage and Redis as the queue backend — the recommended production setup.

## What's included

- **n8n main instance** — serves the editor UI and webhooks
- **PostgreSQL** — stores workflows, credentials, and executions
- **Redis** — powers queue mode for reliable, scalable execution

## First steps

1. Open the generated domain and create your owner account.
2. Set your webhook URL under **Settings → Instance** if you use a custom domain.
3. Import a template from the n8n library or start a workflow from scratch.
