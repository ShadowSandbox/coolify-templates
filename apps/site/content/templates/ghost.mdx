## Overview

Ghost is a powerful publishing platform built for modern creators. Write posts, send newsletters, and run paid memberships — all from one self-hosted app.

This template deploys Ghost with a MySQL database and persistent storage for content and images.

## First steps

1. Open `https://your-domain/ghost` to create your admin account.
2. Configure your mail provider under **Settings → Email newsletter** to send newsletters.
3. Pick a theme or install a custom one from the marketplace.
