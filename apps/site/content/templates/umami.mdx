## Overview

Umami is a simple, fast, privacy-focused web analytics tool. It collects only the metrics you care about, is fully GDPR compliant, and requires no cookie banner.

This template deploys Umami with a PostgreSQL database.

## First steps

1. Log in with the default credentials (`admin` / `umami`) and change the password immediately.
2. Add your website under **Settings → Websites**.
3. Copy the tracking snippet into your site's `<head>`.
